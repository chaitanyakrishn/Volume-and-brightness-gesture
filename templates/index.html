Key Features

Volume Control

Control system volume using right hand gestures. Pinch your thumb and index finger to adjust volume levels.

Brightness Control

Adjust screen brightness using left hand gestures. Perfect for quick adjustments without touching controls.

Mouse Control

Move your mouse cursor using hand positioning. Enable touchless computer control.

Installation Guide

Prerequisites

  • Python 3.7 or higher
  • Webcam
  • Windows/Linux/MacOS

Step 1: Install Required Packages

pip install opencv-python mediapipe pycaw comtypes numpy screen-brightness-control PyQt5 pyautogui

Step 2: Download the Application

Clone the repository or download the source code from our GitHub repository.

Step 3: Run the Application

python gesture_control.py

Usage Guide

Volume Control

  • Use your right hand
  • Pinch thumb and index finger
  • Adjust distance to control volume

Brightness Control

  • Use your left hand
  • Pinch thumb and index finger
  • Adjust distance to control brightness
